# 权威资料来源

最后更新：2026-04-30

本文件用于约束学习资料来源。FinTech 涉及金融、技术、监管、数据和安全，很多内容会随时间变化。遇到政策、监管、API、考试认证、市场数据、产品规则时，应回到官方或专业来源查证。

## 使用规则

- 优先使用一手来源：监管机构、国际组织、专业协会、官方开发者文档。
- 对最新信息，要记录资料发布日期或访问日期。
- 对无法确认的内容，标记为“待查证”。
- 不使用 AI 生成内容作为权威来源。
- 不把博客、论坛、社媒、营销文章作为主要依据。

- Federal Reserve：支付系统、银行、货币政策、金融稳定，https://www.federalreserve.gov/
